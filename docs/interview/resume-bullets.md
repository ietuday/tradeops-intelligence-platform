## Project Summary Paragraph

TradeOps Intelligence Platform is a local enterprise-style trading intelligence system that demonstrates backend architecture, event-driven microservices, observability, and production-readiness practices. The platform includes an API Gateway, identity service, market data ingestion, order and portfolio services, strategy and risk analytics, surveillance alerting, notification delivery, PostgreSQL, Redis, Mosquitto, Redpanda/Kafka, Prometheus, Grafana, Docker Compose, smoke tests, demo scripts, and interview-ready documentation.
