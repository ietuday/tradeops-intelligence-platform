# v1.3.0 Audit Trail & Compliance Reporting

## Added

- Added `audit-service`, a Go microservice for platform audit trails and compliance-style event review.
- Added PostgreSQL audit log and export request tables.
- Added Kafka consumption for user, order, portfolio, risk, surveillance, and notification events.
- Added `audit.log.created` publishing after audit log persistence.
- Added searchable audit APIs, summary API, and JSON/CSV export API.
- Added retry/backoff, idempotency, and `audit.dlq` support.
- Added audit Prometheus metrics and API Gateway `/api/audit` routing.
- Added Docker Compose, Prometheus, smoke test, demo script, sample payloads, and documentation updates.

## Compatibility

- No breaking API changes.
- No frontend, Kubernetes, or Helm work was added.
