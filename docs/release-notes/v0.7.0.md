# v0.7.0 - Risk Engine AI/ML

## Summary

v0.7.0 adds a Python FastAPI risk engine service for portfolio risk score, volatility, drawdown, Value at Risk, anomaly detection, and risk recommendations.

## Included

- Added `services/risk-engine-service` with FastAPI, Pandas, NumPy, scikit-learn, SQLAlchemy, PostgreSQL, Redpanda publishing, JWT/RBAC, and Prometheus metrics.
- Added risk tables for `risk_scores`, `risk_events`, `risk_recommendations`, `risk_anomalies`, and `risk_calculation_runs`.
- Added risk score calculation from concentration, volatility, drawdown, cash buffer, anomalies, and unrealized loss.
- Added symbol-level volatility, portfolio drawdown, parametric/fallback VaR, z-score anomaly detection, and recommendation generation.
- Added API Gateway routes under `/api/risk` with explicit allowlisted proxy mapping.
- Added Docker Compose, `.env.example`, and Prometheus integration for the risk engine service.
- Added smoke coverage for risk health, readiness, score, VaR, and recommendations.

## Not Included

- Notification service
- AI assistant bot
- Kubernetes deployment

## Event Topics

- `risk.score.updated`
- `risk.breached`
- `risk.anomaly.detected`
- `risk.recommendation.created`

## Local URL

- Risk engine service direct URL: `http://localhost:8089`
- API Gateway risk base path: `http://localhost:8080/api/risk`
