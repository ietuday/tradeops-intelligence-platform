# v2.9.0 - Real-Time Dashboard Frontend Enhancement

v2.9.0 adds a lightweight React dashboard for demoing backend platform capabilities through existing APIs and WebSocket streams.

## Added

- React/Vite dashboard under `frontend/trading-dashboard-react`.
- Token paste/save/clear panel using localStorage.
- API client with auth, tenant, and correlation headers.
- WebSocket client with stream selection, simple reconnect, heartbeat visibility, and last-50 event memory.
- Dashboard home page with platform, alerts, notifications, DLQ, rule, and service health summaries.
- Admin Ops page for health, services, topics, DLQs, platform config, and checklist links.
- Risk Analytics page for built-in scenarios and sample stress/concentration/drawdown/volatility calls.
- Observability page linking Grafana, Jaeger, and Prometheus.
- Frontend docs and dashboard demo guide.
- Safe demo helper: `scripts/demo-dashboard.sh`.

## Notes

- Angular shell remains unchanged.
- No Redux, heavy UI library, or backend behavior changes were added.
- The dashboard uses existing API Gateway, admin, risk analytics, and WebSocket endpoints.
