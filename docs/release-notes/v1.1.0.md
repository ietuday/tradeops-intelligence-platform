# v1.1.0 CI/CD, Security Scanning & Quality Gates

## Highlights

- Added GitHub Actions CI workflow for API Gateway tests, Go service tests, Python service tests, script validation, and Docker Compose config validation.
- Added security workflow with secret scanning, Go vet, govulncheck, Node dependency audit, optional Python dependency audit, and basic secret-pattern checks.
- Added Docker workflow to build all service images with local `:ci` tags without pushing to a registry.
- Added docs workflow to validate required documentation and run non-blocking markdown linting.
- Added a root Makefile with local commands for tests, script validation, Compose config, demos, Docker builds, and cleanup guidance.
- Added quality gates documentation under `docs/ci-cd/quality-gates.md`.

## Compatibility

- No breaking changes.
- No business functionality was added.
- No runtime service behavior was changed.
- No Kubernetes or Helm work was added.
