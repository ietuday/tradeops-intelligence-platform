# v0.8.1 - Surveillance Demo & Stabilization

## Summary

v0.8.1 improves demo readiness and documentation for the v0.8.0 surveillance service.

## Included

- Added sample surveillance event payloads under `docs/examples/surveillance`.
- Added `scripts/demo-surveillance.sh` for a local LargeOrderRule demo and alert lifecycle walkthrough.
- Updated README with a dedicated "Demo: Trade Surveillance & Alerting" section.
- Updated smoke tests with direct surveillance-service health and API Gateway surveillance proxy checks.

## Sample Payloads

- `order-created-large-order.json`
- `order-created-normal-order.json`
- `order-cancelled.json`
- `risk-score-updated-breach.json`
- `market-tick-price-jump.json`

## Known Limitations

- `portfolio.updated` and `strategy.signal.generated` are consumed by surveillance-service but do not trigger rules yet.
- The demo script publishes Kafka events through the Redpanda container when Docker Compose is available; otherwise it prints manual publish commands.
