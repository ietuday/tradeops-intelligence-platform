# v0.6.0 - Strategy Studio And Backtesting

## Summary

v0.6.0 adds a Python FastAPI strategy service for creating trading strategies, running local backtests against persisted market ticks, storing performance and signals, and publishing strategy events to Redpanda.

## Included

- Added `services/strategy-service` with FastAPI, Pandas, NumPy, SQLAlchemy, PostgreSQL, Redpanda publishing, JWT/RBAC, and Prometheus metrics.
- Added strategy tables for `strategies`, `backtest_runs`, `strategy_signals`, and `strategy_performance`.
- Added backtest support for `MOVING_AVERAGE_CROSSOVER`, `RSI`, and `VOLATILITY_BREAKOUT`.
- Added API Gateway routes under `/api/strategies` with explicit allowlisted proxy mapping.
- Added Docker Compose, `.env.example`, and Prometheus integration for the strategy service.
- Added smoke coverage for strategy health, readiness, strategy creation, backtest execution, performance, and signals.

## Not Included

- Risk engine
- Notification service
- AI assistant bot
- Kubernetes deployment

## Event Topics

- `strategy.signal.generated`
- `strategy.backtest.completed`

## Local URL

- Strategy service direct URL: `http://localhost:8088`
- API Gateway strategy base path: `http://localhost:8080/api/strategies`
